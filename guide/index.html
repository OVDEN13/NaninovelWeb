# Introduction

Naninovel is an extension for Unity game engine. It consists of a C# framework and editor utilities to support development of visual novel games.

# Features

Following are some of the features that are already implemented; more will be added as the engine undergoes an active development:

  • Document-based scripts
    • Generic text lines
    • Labels
    • Script defines
    • Commands inlining
    • Visual editor
    • IDE support (syntax highlighting, auto-completion, etc)
  • Text printers
    • Dialogue
    • Fullscreen
    • Chat
    • Bubble
    • TextMesh Pro support
    • Ruby (furigana) text support
  • Characters
    • Sprite characters
    • Diced sprite characters
    • Generic characters
    • Live2D characters
    • Character-specific message colors
    • Lip sync
  • Backgrounds
    • Sprite backgrounds
    • Video backgrounds
    • Generic backgrounds
    • Scene backgrounds
  • Background transition effects
  • Perspective camera and look around mode
  • Special effects (FX system)
  • Background music (BGM)
  • Sound effects (SFX)
  • Camera animations
  • Voicing and auto voicing
  • Movies
  • Choices
  • Custom variables
  • State rollback
  • Script expressions
  • In-game variable input
  • Conditional script flow
  • Save-load system
  • Dynamic resources (memory) management
  • Game settings
  • CG gallery
  • Unlockable Tips
  • Cross-platform input
  • Printer backlog
  • Text skipping
  • Auto-advance text
  • UI toggling
  • Adaptive UI layout
  • UI customization
  • Managed text
  • Localization
    • Scripts localization
    • Resources localization
  • Community modding
  • Development console
  • Script rewinding and debug
  • Custom commands
  • Custom actor implementations
  • Google Drive integration
  • Visual scripting
